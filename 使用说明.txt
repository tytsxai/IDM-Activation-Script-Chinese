IDM 激活脚本使用说明
====================

关于本仓库
--------
本仓库提供中文优化的 IDM 激活脚本，批处理与本文档均为 GBK 编码、CRLF 换行，直接在 Windows CMD 中查看不会乱码。

文件一览
--------
IAS.cmd           ―― 主脚本，提供冻结/激活/重置等全部功能
快速激活.cmd      ―― 一键调用 IAS.cmd /frz（默认方案）
测试脚本.cmd      ―― 环境检测脚本（管理员/PowerShell 语言模式/Null 服务/网络/代码页/IAS 路径）
README.md         ―― 详细图文说明
使用说明.txt      ―― 当前文档，用于快速阅读

准备步骤
--------
1. 右键以管理员身份运行“测试脚本.cmd”，确认管理员权限、PowerShell 语言模式、Null 服务、网络、代码页和 IAS 路径均正常。
2. 右键以管理员身份运行“快速激活.cmd”，默认执行冻结激活模式并会自动请求管理员权限。
3. 需要其它操作（普通激活/重置等）时，以管理员身份运行“IAS.cmd”并按菜单选择。

常见问题
--------
- 如脚本显示中文乱码，请在 CMD 中执行 `chcp 936` 再运行脚本。
- 无法联网时，请确认能够访问 https://www.internetdownloadmanager.com（可尝试关闭代理/VPN）。
- 激活后仍提示注册，可先执行“重置激活/试用期”，重启电脑后重新激活。
- 若提示 Null 服务未运行或 PowerShell 语言模式被限制，请先修复对应项后再执行激活脚本。
