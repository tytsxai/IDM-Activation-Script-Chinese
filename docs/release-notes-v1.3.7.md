# IDM 激活脚本中文版 v1.3.7 发布说明

- 版本：**v1.3.7**（2026-06-14）
- 类型：文案/文档更新（脚本逻辑与 v1.3.6 一致）
- 发布包：`release/IDM-Activation-Script-v1.3.7.zip`

## 本次变更

按用户**当前状态**细化"该选哪个激活模式"的说明，避免新手选错：

| 你的状态 | 选哪个 |
|---|---|
| 没领过 30 天试用期 / 想直接能用 | `[2]` 激活（最常用） |
| 已领取并正在使用 30 天试用期（有账号、点过"开始试用"） | `[1]` 冻结（把当前试用期冻住，不让它到期） |
| 选 `[2]` 激活后 IDM 仍提示"未注册" | 改用 `[1]` 冻结兜底 |

README、`使用说明.txt`、`llms.txt` 均已同步该说明；`IAS.cmd` 版本号升至 1.3.7。

## 与 v1.3.6 的关系

v1.3.6 已修复"目录不可写"误报、`(x86)` 路径提权崩溃、Win11 WMI 误报、激活过程中文乱码，并把四个入口脚本合并为单一 `开始激活.cmd`。v1.3.7 仅在其基础上更新文案与版本号，**脚本行为不变**，可直接覆盖升级。
