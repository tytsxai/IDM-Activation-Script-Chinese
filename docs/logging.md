# 日志与静默模式说明

- 参数：`/silent`（或 `/quiet`）启用静默执行，自动开启日志；`/log` 仅开启日志。静默模式未指定 `/act`/`/frz`/`/res` 时会直接退出并返回码 2。
- 日志：输出到 `%SystemRoot%\\Temp\\IAS-<时间戳>.log`，时间戳源自本机时间；记录启动参数、注册表备份与增删、下载校验结果、退出码等关键节点。
- 返回码：`0` 成功；`1` 操作失败（如未安装 IDM、下载校验失败、注册表写入失败）；`2` 环境或参数错误（如缺少管理员/WMI/Powershell、路径/行尾异常、静默缺少操作参数）。
- 行为：默认交互保持不变；静默模式下跳过菜单/暂停，等待时间缩短；开启日志时会在控制台提示日志路径（静默模式仅写入文件）。
- 清理：日志存放在系统临时目录，如需关闭或清理可手动删除对应文件。
